Human-dates¶

This package is the result of the repackaging of the original human-dates, that it’s coming itself from a Stack Overflow answer citing a further source, not available anymore.

Even if the original function it’s quite trivial there is no need (nor chance) for a smarter implementation, and it’s quite comfortable to have it packaged and developed separately in most of the situations (where you don’t want to maintain a lengthy barely useful function, but that you would like to have the same if provided by someone else).
